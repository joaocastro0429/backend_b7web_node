HTTP (HyperText Transfer Protocol) é um protocolo de comunicação usado para a transferência de dados entre um cliente (como um navegador) e um servidor na web. 

HTTPS (HyperText Transfer Protocol Secure) é a versão segura do HTTP, que utiliza criptografia para proteger a comunicação entre o cliente (como um navegador) e o servidor. Ele usa SSL/TLS para garantir que os dados transmitidos não sejam interceptados ou alterados por terceiros.

FTP (File Transfer Protocol) é um protocolo de rede usado para transferir arquivos entre um cliente e um servidor. Ele permite o envio, recebimento e gerenciamento de arquivos em um servidor remoto, sendo bastante utilizado para hospedagem de sites e backups.

SMTP (Simple Mail Transfer Protocol) é um protocolo de comunicação usado para o envio de e-mails na internet.

DNS(Domin Name System) - O DNS (Sistema de Nomes de Domínio) é um sistema que traduz nomes de domínio (como google.com) para endereços IP (como 142.250.217.14), permitindo que os usuários acessem sites sem precisar memorizar endereços numéricos.
